# ⚒️power-toys-spring-boot-starter
本项目的目标是通过SpringBoot的spring.factories机制实现组件的快速集成。

## 🎯计划
预计支持一下组件

| component  | name      | Links                                            |
|------------|-----------|--------------------------------------------------|
| DataSource | MySql     |                                                  |
|            | MongoDB   |                                                  |
|            | Redis     |                                                  |
|            | Liquibase | https://www.liquibase.org/get-started/quickstart |
|            | Flyway    |                                                  |
| Swagger    | Knife4j   | https://doc.xiaominfo.com/                       |
| rpc        | nacos     |                                                  |
|            | dubbo     |                                                  |
| gRPC       |           |                                                  |
